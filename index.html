HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title within the head element displays in the website tab
  • h1 through h6 represent different sizes of headers from large to small

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • https://htmlcolorcodes.com/ can help you select a different HEX color codes to use
  • If you set a class within an element you can then call it in the style.css file

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch: tells you the branch you are currently on and shows other branches available
  • git reflog: will list everythig done in git across all of your branches
  • git add -A: to stage all files in the working branch
  • git commit -m "-m is message": will commit the previously staged files
  • git pull origin main: will make sure the current branch is up to date before making a pull request
  • git push origin 'branch name': pushes up the changes from the branch

JavaScript

  • console.log: Allows you to print out to the 'console' can be viewed by inspecting the page in your browser.
  • Array: Allows you to hold a group of related data. Does not have to be the same type of data. Can hold a sting + number + boolean all in the same array.
  • Control Flow: The order a computer executes code.
  • For Loop: Allows you to iterate over an array.
